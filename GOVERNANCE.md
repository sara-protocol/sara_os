# Governance

SARA_os is governed as cognitive infrastructure, not as a product or service.

---

## Core Rules

1. Human sovereignty overrides all optimization goals.
2. Frozen versions cannot be silently modified.
3. Any advisory or decision-substitution use is misuse.

---

## Change Policy

All changes require:
- Version increment
- Public changelog
- Explicit scope description

---

## Misuse Clause

Using SARA_os to:
- manipulate decisions,
- induce dependency,
- replace judgment,

constitutes protocol violation.
